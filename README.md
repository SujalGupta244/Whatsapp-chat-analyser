# Steps to run the program
run the following commands
``` bash 
cd myproject
```

``` bash 
python -m venv .venv
```

``` bash 
# macOS and Linux
source .venv/bin/activate
```
``` bash 
pip install streamlit
```

Create a file named app.py in your project folder.

``` bash 
import streamlit as st

st.write("Hello world")
```
